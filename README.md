# MATLAB app for extracting, reformatting and uploading C-POD and F-Pod files

## Project description
This project is part of the EU DTO-BioFlow (https://dto-bioflow.eu/). The focus of this particular project is to aggregate passive acoustic monitoring (PAM) data, and harmonise and standardise it. Specifically, the data is harbour porpoise detections that have been collected with C-PODs or F-PODs.
